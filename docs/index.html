HIL: Closer to Reality

Why HIL?

HIL Simulation validates a control system's design by running the control on its actual hardware, making the testing environment closer to reality.

Testbed Structure

  • Testbed Overview
  • System Data Flow
  • Simulator: Models the inverters, measurements, and the power grid in real time
  • OP8666: DSP Controller
  • Monitor and Oscilloscope: Data Monitoring

Control Structure

  • Communication: Bidirectional communication using two sets of transmitters and receivers. Time step: 0.025s.
  • Cores Assignment:
    • Core1: Primary control, secondary control, and clockwise communication.
    • Core2: Counterclockwise communication.
    • Communication between Core1 and Core2 via global RAM.
